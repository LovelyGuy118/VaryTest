# Vary Assessment

## About App

An applicaton used to analyze imports and contracts from solidity code

## Technologies

Use Node.js, Express, typescript

## Installation

### Prerequisites

Node version 16.14.2

### Run

Installation:

`npm install`

To Start Server:

`npm start`

## Guide for Usage

Run server

Send post request using Postman or other tools

API Endpoint is /analyze
